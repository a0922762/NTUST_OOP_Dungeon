## 按下Space進入地宮按下Q離開


## 角色基本控制說明
### WASD: 上下左右
### Space: 攻擊 / 使用items
### Ctrl + s: 儲存
### Ctrl + l:加載
### Ctrl + up: 音量調大
### Ctrl + down: 音量調小
